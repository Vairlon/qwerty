Отдыхатель пальчиков

Возможности программы:

  • Настройка двух разных позиций для кликов
  • Установка задержки для каждой позиции в минутах и секундах
  • Удобное управление через горячие клавиши (F6, F7, F8)
  • Простой и понятный интерфейс

Как использовать:

  1. Скачайте программу
  2. Запустите AutoClicker.exe
  3. Нажмите F6 чтобы установить первую позицию
  4. Нажмите F7 чтобы установить вторую позицию
  5. Установите желаемые задержки
  6. Нажмите F8 для запуска/остановки кликера
